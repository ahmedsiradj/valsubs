## valsubs

It's a python tool that used to validate subdomains 

### Installation:

```bash
git clone https://github.com/ahmedsiradj/valsubs.git
cd subscollector/
pip3 install -r requirements.txt
```

```bash
python valsubs.py -h
```

### Usage:

```bash
python valsubs.py -i subdomains.txt -t 0.10
```
